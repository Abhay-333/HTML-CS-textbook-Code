        The difference between div & span
    1. By default, div takes the whole width of its parent whereas the span takes the width of the content or element within it.
    2. Div is a block level element and span is an inline element. 
    3. The div should be used to wrap sections of a document, while the span should be used to wrap small portions of text, images, etc.

    3 ways to add css:
    
    a.Inline css - It is directly written in the html element like style attribute.It is not recommended to use inline css for huge production website because its time comsuming.We use it to check a quick output.

    b.Internal css - If we had a website of 100 of pages 1 of the pages requires unique design i.e different from rest of the webpages. In this case we use internal css means adding css code in the head section of html code.We use style tag for this.

    c.External css - If we have 100 of html files in our project and we want similar design for all the 100 file. For this external css is used.
